
# Linux Commands – Foundations

This document tracks Linux commands learned during the Month 0 (january 2026) of my cybersecurity roadmap,
with a focus on security and system fundamentals.


## Navigation & File Management
- `pwd` – show current directory
- `ls` – list files
- `ls -a` – list all files including hidden ones
- `ls -la` – list all files with detailed information
- `cd` – change directory
- `tree` – display directory structure

## File Inspection & Content Analysis
- `cat` – read file contents
- `less` – view large files
- `file` – identify file type
- `strings` – extract readable strings from binaries
-  `base64` - encode or decode Base64 data
## Search & Filtering
- `find` – search files by name, size, owner, permissions
- `grep` – search text inside files
- `grep -R` – recursive search
- `sort` – sort output
- `uniq` – filter duplicates

## Permissions & Ownership
- `ls -l` – view permissions of files

## Pipes & Redirection
- `|` – pipe output
- `>` – redirect output
- `>>` – append output
- `2>/dev/null` – suppress errors

## Networking & Remote Access
- `ssh` – secure remote login
- `scp` – copy files over SSH

## Processes & Environment
- `ps` – view running processes
- `top` – monitor processes
